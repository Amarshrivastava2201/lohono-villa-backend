# AI Usage Documentation

## Tools Used
- ChatGPT (OpenAI)

## How AI Was Used
- Project structure planning
- API contract clarification
- Data modeling guidance
- Validation of business rules

## What Was Accepted
- Folder structure suggestions
- Date handling logic
- Pagination and sorting approach

## What Was Rejected
- Over-optimized aggregation pipelines
- Unnecessary abstractions
- Complex frontend state management

## What Was Modified
- Availability logic rewritten for clarity
- Error handling simplified for readability
- Query logic adjusted to match assignment constraints

## Reasoning
The focus was on correctness, clarity, and alignment with the assignment requirements rather than premature optimization.
